1.只有 null undefined '' NaN 0 false 这几个是 false，其他的情况都是 true，比如 {} , []。 2.undefined 转换为数字是 NaN，['5']首先会变成 '5', 然后再变成数字 5 使用 == 比较中的规则： 1.NaN和其他任何类型比较永远返回false（包括和他自己） 2.Boolean 和其他任何类型比较，Boolean 首先被转换为 Number 类型。 3.String和Number比较，先将String转换为Number类型。 4.null == undefined比较结果是true，除此之外，null、undefined和其他任何结果的比较值都为false。
